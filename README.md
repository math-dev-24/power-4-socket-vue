# Installation

## Cloner le projet

```bash
    git clone https://github.com/mathieuprog/power-4-socket-vue.git
    cd power-4-socket-vue
```

## Frontend

```bash
    cd pwer-4
    npm install
    npm run dev
```

## Backend

```bash
    cd pwer-4-back
    npm install
    npm run dev
```
